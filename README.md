# Placement Management System

A web-based Placement Management System developed using **Spring Boot, Thymeleaf, and MySQL**.  
The application helps administrators manage student records, company details, and placement status through an interactive dashboard.

## Features

- Admin Login Authentication
- Student Management
- Company Management
- Placement Assignment
- Dashboard with Placement Statistics
- Pie Chart Visualization of Placements
- Recent Placements Display

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- Bootstrap
- Chart.js
